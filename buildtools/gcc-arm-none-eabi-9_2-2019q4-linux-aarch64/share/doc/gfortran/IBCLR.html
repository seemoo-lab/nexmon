9.140 IBCLR — Clear bit

Description:

IBCLR returns the value of I with the bit at position POS set to zero.

Standard:

Fortran 95 and later, has overloads that are GNU extensions

Class:

Elemental function

Syntax:

RESULT = IBCLR(I, POS)

Arguments:
IThe type shall be INTEGER.
POSThe type shall be INTEGER.
Return value:

The return value is of type INTEGER and of the same kind as I.
