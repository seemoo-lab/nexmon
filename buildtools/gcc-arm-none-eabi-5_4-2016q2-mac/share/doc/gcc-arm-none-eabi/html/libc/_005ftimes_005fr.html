12.2.14 _times_r—Reentrant version of times

Synopsis 

     #include <reent.h>
     #include <sys/times.h>
     clock_t _times_r(struct _reent *ptr, struct tms *ptms);

Description
This is a reentrant version of times. It takes a pointer to the global data block, which holds errno.
