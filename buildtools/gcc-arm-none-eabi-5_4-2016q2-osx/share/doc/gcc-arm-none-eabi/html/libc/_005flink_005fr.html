12.2.6 _link_r—Reentrant version of link

Synopsis 

     #include <reent.h>
     int _link_r(struct _reent *ptr,
         const char *old, const char *new);

Description
This is a reentrant version of link. It takes a pointer to the global data block, which holds errno.
