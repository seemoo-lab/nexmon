12.2.11 _kill_r—Reentrant version of kill

Synopsis 

     #include <reent.h>
     int _kill_r(struct _reent *ptr, int pid, int sig);

Description
This is a reentrant version of kill. It takes a pointer to the global data block, which holds errno.
