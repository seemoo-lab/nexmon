12.2.7 _lseek_r—Reentrant version of lseek

Synopsis 

     #include <reent.h>
     off_t _lseek_r(struct _reent *ptr,
         int fd, off_t pos, int whence);

Description
This is a reentrant version of lseek. It takes a pointer to the global data block, which holds errno.
