Prerequisites

GLoadableIcon requires GIcon and GObject.

Known Implementations

GLoadableIcon is implemented by GBytesIcon and GFileIcon.

Includes

#include <gio/gio.h>

Description

Extends the GIcon interface and adds the ability to load icons from streams.
